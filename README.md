## How to Run

To run the Hello World program:

1. Clone this repository:
```bash
git clone https://github.com/ryanemmanueladam/Hello-World.git
```

2. Navigate to the project directory:
```bash
cd Hello-World
```

3. Run the Python script:
```bash
python hello_world.py
```
